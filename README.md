# Calc24Game

## 介绍

在线联机进行24点游戏

## 软件架构

服务端部署在linux中，客户端暂时以nc进行模拟


### v1版本

一个连接开启一个线程模型
产生了很多问题：

* 性能上不行（连接数越多，线程越多，线程会不断切换，会浪费时间）
* 设计上需要为每个连接的各种数据结构增加一个锁，代码非常难维护（频繁出现多个线程操作一个fd的事情，就要加锁）
* 业务与框架耦合比较严重

### v2版本：one thread one loop模型


### 目前实现的功能：

#### 1、联机游戏

分桌，当一个桌子有三个人在并准备即可发同一牌，最先算出结果的获胜，并变成空闲状态

#### 2、用户互动

同一桌的可以任意交流

#### 3、服务器反馈

让玩家准备、答对反馈、答错反馈
